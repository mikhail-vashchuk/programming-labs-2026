# Movie Structure Program

Program for working with a `Movie` structure in C++.

## Features
- show all movies
- search movies by director
- show movies with profit greater than a given value and shot in Hollywood
- sort movies by title in alphabetical order

## Structure fields
- title
- director
- country
- location
- profit

## Purpose
Practice working with:
- structures
- arrays of structures
- `char` arrays
- search
- sorting
- input validation

## Build & Run (macOS/Linux)

```bash
cmake -S . -B build
cmake --build build
./build/app
