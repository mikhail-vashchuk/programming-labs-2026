# Text File Lab

Reads passenger data from a file, processes it, and writes a report.

## Features
- Passengers with ≤ 2 items
- Average items count
- Count of passengers above average

## Files
- generateFile.cpp — creates input file
- main.cpp — processes data

## Build & Run (macOS/Linux)

```bash
cmake -S . -B build
cmake --build build
./build/app
