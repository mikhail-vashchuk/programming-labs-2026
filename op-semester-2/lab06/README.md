# lab06

Multi-file C++ project based on lab05.

## Structure
- `src/main.cpp` — menu and main program flow
- `src/state.h` — `State` structure and function declarations
- `src/state.cpp` — function implementations
- `CMakeLists.txt` — build configuration

## Features
- create and delete binary file
- add and remove records
- show all records
- show countries with more than one official language
- show countries with a USD rate below a given value
- sort countries alphabetically
